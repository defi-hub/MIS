## Usage Notes / 使用说明 / 使用上の注意 / 사용 참고사항

### Translation Guidelines / 翻译指南 / 翻訳ガイドライン / 번역 가이드라인

1. **Technical Terms**: Keep English abbreviations (e.g., eBPF, LSM) in parentheses after translation
   - 技术术语：在翻译后的括号中保留英文缩写
   - 技術用語：翻訳後の括弧内に英語の略語を保持する
   - 기술 용어: 번역 후 괄호 안에 영어 약어 유지

2. **Consistency**: Always use the same translation for the same term
   - 一致性：始终对同一术语使用相同的翻译
   - 一貫性：同じ用語には常に同じ翻訳を使用する
   - 일관성: 동일한 용어에 항상 동일한 번역 사용

3. **Context Matters**: Some terms may have different translations depending on context
   - 上下文很重要：某些术语可能因上下文而有不同的翻译
   - 文脈が重要：文脈によって異なる翻訳がある場合がある
   - 맥락이 중요: 일부 용어는 맥락에 따라 다른 번역을 가질 수 있음
